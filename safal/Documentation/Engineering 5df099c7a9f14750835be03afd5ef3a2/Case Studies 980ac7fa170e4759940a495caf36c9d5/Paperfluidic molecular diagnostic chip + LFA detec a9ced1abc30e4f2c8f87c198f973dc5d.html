Paperfluidic molecular diagnostic chip + LFA detection + Electrical Heating

Device Setup -Production (Origami-like method):

Equipment Used + Procedure:

Equipment used:

  • Lysed sample in a tube
  • Standard letter size self-adhesive laminating sheets (Fellowes product # 5221502) for base
  • Polyethersulfone (PES) filter paper (Millipore, cat# GPWP04700) for sample pad
  • Cellulose blotting paper (Whatman GB003, cat# 09-301-400) for absorbent pad
  • Universal LFD Strip
  • Electrical Heating Block
  • Pipetting Equipment

For procedure, see figure 2:

Key Findings

  • Very cheap, relatively easy to make platform that is also readily disposable
  • Use of standard products with minimal fabrication
  • Extracts DNA using standard filter paper techniques
  • Cons: Requires manual fluid transfer using pipettes, cannot be easily multiplexed, some reagents need cold storage
